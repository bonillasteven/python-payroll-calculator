
# Python Payroll Calculator

A Python payroll calculator that computes:

- Gross Pay
- Overtime Pay
- Federal Tax
- State Tax
- Social Security
- Medicare
- Net Pay
- Annual Salary Estimates

Built using Python functions, loops, conditionals, and formatted output.

## Features
- Overtime calculation
- Payroll deductions
- Annual gross/net salary
- Repeating calculations with loops

## Technologies
- Python
